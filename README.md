# python-mini-project

A collection of beginner-friendly Python programs created for learning basic programming concepts.

## Projects Included

1. Calculator
2. Password Generator
3. Number Guessing Game
4. Temperature Converter
5. BMI Calculator

## Technologies Used

- Python 3

## How to Run

Clone the repository:

```bash
git clone https://github.com/impana-p21/python-mini-projects.git
```

Run any project:

```bash
python calculator.py
```

Replace `calculator.py` with any other file.
